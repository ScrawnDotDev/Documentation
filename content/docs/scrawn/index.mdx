# Scrawn — The Anti-Billing Billing Tool

Scrawn is the **open-source usage engine** for developers who wants to charge for usage. Every API call, AI token, or database write, without writing an entire billing platform.

<Callout type="success" title="Dodo Gets the Cash. We Do the Math.">
We aggregate usage, evaluate your pricing logic, and hand you a payment link in a single function call. You keep full control of your Dodo Payments account.
</Callout>

## What is Scrawn?

Scrawn is the **missing aggregation layer** for [DodoPayments](https://dodopayments.com). Instead of wiring up gRPC, PostgreSQL, idempotency keys, auth hashing, and pricing resolvers yourself, you call a single SDK function:

```typescript
import { scrawn, tag } from "@scrawn/core";

const biller = scrawn({
  apiKey: process.env.SCRAWN_KEY as `scrn_${string}`,
  baseURL: process.env.SCRAWN_BASE_URL || "http://localhost:8069",
});

// Track a billable event
await biller.basicUsageEventConsumer({
  userId: "user-123",
  debit: tag("TAG_NAME"),
});

const billingLink = await biller.collectPayment("user-123");
```

<Callout type="success">
**One function call** — no proto files, no DB migrations, no webhook nightmares.
</Callout>

## The Scrawn Way

**Before Scrawn** — 60+ lines of boilerplate you didn't write (but now have to maintain):

<Callout type="warn">
You'd need: gRPC client setup, proto type generation, DB client + schema, auth hashing + DB lookup, idempotency (HMAC + Redis), tag resolution or expression evaluation, debit validation, DB transactions, error handling... and that's before you even think about AI token billing.
</Callout>

**With Scrawn** — one function call:

```typescript
import { scrawn } from '@scrawn/core';

const biller = scrawn({ apiKey: env.SCRAWN_KEY });

await biller.basicUsageEventConsumer({
  userId: body.userId,
  debit: 50,
});
```

## Use Cases

- **AI/LLM Services** — bill per token per model with automatic Vercel AI SDK integration
- **API Platforms** — charge per request with tag-based pricing tiers
- **SaaS Products** — usage-based pricing for any metric
- **Dev Tools** — metered access to CLI tools or build pipelines

## Features

- **DodoPayments Native** — we plug into your existing Dodo account; no Stripe migration needed
- **AI Token Metering** — first-class support for Vercel AI SDK (`streamText`, `generateText`, `streamObject`, `generateObject`)
- **Pricing DSL** — `mul()`, `add()`, `sub()`, `tag()` — type-safe expressions for any pricing model
- **Middleware Mode** — intercept requests, extract user IDs and costs, bill automatically
- **Self-Hosted** — your DB, your data, your infra. Docker compose included.
- **gRPC Backend** — high-performance, stateless, horizontally scalable
- **Type-Safe SDK** — full TypeScript types for events, errors, and configuration
- **Framework Agnostic** — Express, Fastify, Hono, Next.js, tRPC, NestJS — any Node.js HTTP framework
