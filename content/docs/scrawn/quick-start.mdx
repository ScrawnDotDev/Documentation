Install the SDK and start tracking billable events — no backend setup or config required.

## 1. Install

<Tabs items={['bun', 'npm', 'yarn']}>
  <Tab value="bun">
    ```bash
    bun add @scrawn/core
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @scrawn/core
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @scrawn/core
    ```
  </Tab>
</Tabs>

## 2. Initialize

```typescript
import { scrawn, tag } from "@scrawn/core";

const biller = scrawn({
  apiKey: process.env.SCRAWN_KEY as `scrn_${string}`,
  baseURL: process.env.SCRAWN_BASE_URL || "http://localhost:8069",
});
```

## 3. Track Basic Usage

Call `basicUsageEventConsumer` whenever a billable action happens:

```typescript
// Direct amount — 500 cents = $5.00
await biller.basicUsageEventConsumer({
  userId: "user-123",
  debit: 500,
});

// Or reference a price tag managed on the backend
await biller.basicUsageEventConsumer({
  userId: "user-123",
  debit: tag("PREMIUM_CALL"),
});
```

## 4. Track AI Token Usage

Stream per-token usage with an async generator:

```typescript
const response = await biller.aiTokenStreamConsumer({
  userId: "user-123",
  model: "gpt-4",
  inputTokens: 100,
  outputTokens: 50,
  inputDebit: 3,   // 3¢ per input token
  outputDebit: 6,  // 6¢ per output token
});
// → { eventsProcessed: 1 }
```

Or wrap the Vercel AI SDK with `biller.ai()` to auto-bill every LLM call:

```typescript
import * as ai from "ai";

const aii = biller.ai(ai, {
  inputDebit: mul(biller.tag("GPT_INPUT"), inputTokens()),
  outputDebit: mul(biller.tag("GPT_OUTPUT"), outputTokens()),
});

const result = await aii.streamText({
  userId: "user-123",
  model: openai("gpt-4o-mini"),
  prompt: "Hello",
});
```

## 5. Collect Payment

When a user's balance crosses your threshold, collect with one call:

```typescript
const checkoutLink = await biller.collectPayment("user-123");
// → "https://checkout.dodopayments.com/..."
```
